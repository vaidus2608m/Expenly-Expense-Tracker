A clean expense tracker built with vanilla JavaScript. Tracks income and expenses, visualizes data with a donut chart, and persists state using localStorage.

Expenly is a minimal expense tracker built using vanilla HTML, CSS, and JavaScript.

The goal of this project is to practice core frontend fundamentals such as state management, DOM manipulation, and data persistence using browser localStorage.

## Features

- Add income and expense transactions
- Automatic balance, income, and expense calculation
- Donut chart visualization
- Data persistence using localStorage
- Clean and minimal UI
- Responsive layout

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser localStorage API

## Project Structure

index.html

script.js  

## Learning Focus

- Separating application state from the DOM
- Persisting and restoring data on page reload
- Dynamic UI updates based on state
- Form handling and validation
- Basic data visualization logic
